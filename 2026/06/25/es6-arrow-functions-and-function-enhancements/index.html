前言

ES6 对函数做了大量增强，让 JavaScript 的函数更简洁、更安全、更强大。

默认参数 手动判断 undefined ✅
剩余参数 arguments 伪数组 ✅
尾调用优化 调用栈溢出 ✅

总结

ES6 函数特性 作用 建议

参数默认值 代替手动 undefined 判断 推荐在所有可选参数使用
