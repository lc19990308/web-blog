前言

TypeScript = JavaScript + 类型系统。它不是一门新语言，而是 JS 的超集——所有 JS 代码都是合法的 TS 代码。

为什么需要 TypeScript？

  • 在编码阶段发现 Bug（而非运行时）
  • 更好的 IDE 智能提示和自动补全
  • 代码自文档化（类型即文档）
