五、性能分析

Data URL vs HTTP URL

场景 Data URL HTTP URL
请求次数 0 次 1 次
文件大小 **+33%**（base64 开销） 原始大小
缓存 ❌ 每次都下载 ✅ 强缓存
首屏 首屏 HTML 变大 首屏 HTML 小
小图标（< 8KB） ✅ 推荐 多一次请求不划算
大图片 ❌ 不推荐 ✅ 推荐

结论： 小图（< 8KB）用 Data URL，大图用 HTTP URL。现代打包工具会自动帮你做这个决策。
