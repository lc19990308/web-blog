前言

JavaScript 在 ES5 中引入了严格模式（Strict Mode）。在严格模式下，一些不安全或容易出错的写法会被禁止，从而写出更健壮的代码。

关键认知：ES Module（type=”module”）和 class 内部默认就是严格模式，无需手动开启。

Q1：严格模式能提升性能吗？

不一定。 某些优化（如省略变量声明检查）在严格模式下更高效，但具体的性能差异微乎其微。严格模式的主要价值是代码安全性。
