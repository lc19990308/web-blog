前言

闭包（Closure）是 JavaScript 最核心的概念之一，也是面试高频题。理解闭包的关键在于理解两个机制的叠加：

闭包 = 函数作为值传递 + 词法作用域

当一个函数被定义时，它会”记住”它所在的词法作用域。即使这个函数在其他地方执行，它依然能访问到定义时的作用域中的变量。
