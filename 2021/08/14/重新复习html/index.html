HTML 语义化与常用标签完全指南

发表于 2021-08-14 分类于 CSS 阅读次数：
掌握 HTML5 语义化标签的正确使用方式，理解 SEO 优化、无障碍访问与常用标签的最佳实践

前言

一个优秀的前端开发者，绝不是靠 <div> 一把梭。HTML5 提供了丰富的语义化标签，不仅能提升 SEO，还能改善无障碍访问体验。

一、为什么语义化？

好处 说明
SEO 搜索引擎能识别内容结构
无障碍 屏幕阅读器可以正确导航
可维护 团队协作一目了然
默认样式 部分标签自带浏览器样式

二、HTML5 常用语义化标签

2.1 页面结构标签

标签 含义 使用场景
<header> 头部区域 页面或文章的页眉
<nav> 导航 主导航、目录
<main> 主要内容 页面独一份的内容
<article> 独立内容块 博客文章、新闻、评论
<section> 章节分组 文章中的各个段落
<aside> 侧边栏 广告、相关链接
<footer> 尾部区域 版权、联系信息

2.2 文本语义标签

标签 含义 浏览器效果
<strong> 重要（粗体） 加粗
<em> 强调（斜体） 斜体
<blockquote> 长引用 缩进
<q> 短引用 带引号
<code> 代码 等宽字体
<time> 时间日期 无特殊样式
<abbr> 缩写 虚线下划线

url URL 手机端弹出 .com 键盘
date 日期 弹出日期选择器
color 颜色 弹出颜色选择器
