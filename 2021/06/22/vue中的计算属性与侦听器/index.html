前言

computed 和 watch 是 Vue 中处理派生状态的两种方式。新手常困惑：什么时候用 computed？什么时候用 watch？

核心原则只有一句话：

computed 用于派生值，watch 用于执行副作用。

无副作用 不应修改外部状态（只计算，不操作）
