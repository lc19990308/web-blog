CSS 中的单位种类繁多，选错了可能导致布局异常、字体不缩放、手机端适配困难。本文一次讲清楚所有常用单位的区别和用法。

五、px——像素

px 的真相： 1px 不等于 1 个物理像素（高 DPI 屏幕下，1px 可能由 2×2 或 3×3 个物理像素渲染）。这就是为什么 1px 在某些屏幕上看起来更细的原因。

什么时候用 px？

  • 边框 border: 1px solid
  • 阴影 box-shadow
  • 小图标尺寸
  • 需要精确像素对齐的场景

七、选型速查表

场景 推荐单位 理由
字体大小 rem 统一缩放，无障碍友好
容器宽度 % 或 vw 响应式
全屏背景 100vh 撑满视口
边框 px 固定不变
间距（padding/margin） rem 或 em 随字体缩放
正方形 vmin 保持比例
混合计算 calc() 灵活组合
移动端适配 vw + rem 常见方案
