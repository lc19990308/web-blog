Biome 入门指南：取代 ESLint + Prettier 的统一工具链

发表于 2025-04-08 分类于 工程化 阅读次数：
Biome 是用 Rust 写的统一前端工具链，一个工具搞定代码检查、格式化、排序导入。本文从安装到配置，对比 ESLint + Prettier 的性能提升

前言

ESLint + Prettier 的组合统治了前端代码规范多年，但有两个痛点：

  1. 慢——JS 写的，大项目 lint + format 要几十秒
  2. 配置复杂——两个工具互相冲突，需要 eslint-config-prettier 调和

Biome 用 Rust 重写了这一切——一个工具搞定 lint、format、sort imports，速度是 ESLint 的 10-30 倍。
