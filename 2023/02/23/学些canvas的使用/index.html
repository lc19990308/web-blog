学习如何使用 canvas。

绘制一条直线。

canvas 有着自己的默认宽高，宽 300，高 150.可以在 canvas 标签上进行设置。如果在 css 里面进行设置的话，会导致 canvas 默认本身的宽高出现拉伸的情况。

默认 canvas，是黑色，线条宽度 1px，但是默认的 canvas 线条在底部，像素之间会堆叠，产生了 2px 的效果。

canvas 的坐标系

canvas 的坐标系问题。canvas 遵守的是，w3c 坐标系，从左往右是 x 轴。从上到下是 y 轴，这个跟数学坐标系是不同的。我们使用 canvas，画 2 条线条。

第二条线，是一条斜线，是因为，y 坐标轴，从 10，30 是一个从上到下的关系。

总结

今天先初步认识一下，canvas 的使用。

canvas 的宽高，需要写在标签里面，或者使用 js 对宽高进行定义，css 设置宽高，会存在拉伸效果。

canvas 使用的是 w3c 坐标系，从上到下，是 y 轴，从左到右是 x 轴。和数学里面的坐标轴不同。

可以通过 canvas 去画一些简单的图形。

canvas 的 api 有。

  • moveTo，代表 canvas 坐标的起点。
  • lineTo，代表下个坐标的位置。
  • stroke，把 canvas 的坐标给连接起来。
  • lineWidth，设置线的粗细，
  • strokeStyle，设置线的颜色
  • lineCap 线帽：默认: butt; 圆形: round; 方形: square
  • beginPath，当我们要设置多条线条样式的时候，必须用它。
  • strokeRect，可以对 canvas 周围进行描边。
  • strokeStyle，设置描边颜色。
  • fillRect，填充到整 canvas，
  • fillStyle，设置填充颜色
  • rect，设置矩形，但是必须通过 fill 来渲染。
  • fill，渲染矩形。
  • clearRect，清除 canvas 上的线段，矩形，或者整个画布。
  • closePath，解决多边形闭合问题。
  • arc，设置圆形。设置圆形必须先使用，beginPath
  • arcTo 画弧线。
  • font 设置文本
  • strokeText 对文本进行描边
  • strokeStyle，设置描边文本颜色
  • fillText，对文本进行填充
  • fillStyle，设置填充文本颜色。
  • 获取文本长度，measureText
  • textAlign，文本对齐方式
  • textBaseline，设置文本垂直方式
  • drawImage，canvas 加载图片，截取图片。
